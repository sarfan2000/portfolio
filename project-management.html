Project Management Platform

Project Overview

A role-based project management platform developed as a group project to streamline task assignment, team collaboration, and project tracking. The platform provides separate dashboards for clients, team members, and administrators with real-time visibility into project progress.

Objectives

  • Implement role-based access control for clients, team members, and admins
  • Enable efficient task creation, assignment, and deadline tracking
  • Provide real-time project progress monitoring for all stakeholders
  • Facilitate team collaboration through structured project workflows
  • Manage client relationships and communication in one platform

Key Features

Role-Based Access Admin, team member, and client permission levels
Task Management Create, assign, and track tasks with priorities
Progress Monitoring Real-time dashboards for project status
Team Management Manage team members and their workloads
Client Management Dedicated client portal and communication
Notifications Alerts for task updates and deadlines

System Architecture

  • Frontend: TypeScript for type-safe, maintainable UI components
  • Backend: Node.js + Express.js API with role-based middleware
  • Database: MongoDB for flexible project and task document storage
  • Auth: JWT tokens with role-based route protection
  • Team Size: Group project with collaborative development workflow

Technologies Used

TypeScript Node.js Express.js MongoDB JWT REST API
