Automated Futsal Smart System

Project Overview

A smart reservation platform built for indoor futsal sports facilities to modernize court booking. The system provides real-time court availability and online reservations through both a web application and a cross-platform mobile app, eliminating phone-based bookings and scheduling conflicts.

Objectives

  • Enable real-time court availability checking for customers
  • Allow online advance booking and reservation management
  • Develop cross-platform mobile app for iOS and Android
  • Build a web-based management dashboard for facility owners
  • Automate time slot management and conflict prevention

Key Features

Real-Time Availability Live court status and open slot display
Online Booking Advance court reservation with instant confirmation
Mobile App Flutter-based cross-platform app for iOS & Android
Web Dashboard React.js management panel for facility owners
Conflict Prevention Automated double-booking detection and prevention
Booking History User booking records and upcoming reservations

System Architecture

  • Mobile App: Flutter (Dart) — single codebase for iOS and Android
  • Web Frontend: React.js for the facility management dashboard
  • Backend: Node.js + Express.js RESTful API shared by both platforms
  • Database: MongoDB for storing courts, slots, and booking records
  • Real-Time: API polling for live availability updates

Technologies Used

Flutter Dart React.js Node.js Express.js MongoDB REST API
